================================================================================
Cloud Shell — deploy Snippd Agent Engine
================================================================================

1) If your prompt is already ~/something, do NOT "cd something" again — you are there.

2) You need these files in the current directory:
      agent.py  deploy.sh  requirements.txt  agents/  tools/
   If `ls` does not show them, you are in the wrong folder (e.g. only main.py and '=ERROR').

3) Clone this repository (after you push it to GitHub from your laptop), then:
      cd ~
      git clone https://github.com/YOUR_ORG/snippd-beta-demo.git
      cd snippd-beta-demo/snippd_agent
      chmod +x deploy.sh verify_tree.sh bootstrap_cloudshell.sh
      ./verify_tree.sh
      gcloud config set project YOUR_PROJECT_ID
      ./deploy.sh

   Folder name must be snippd_agent (underscore), not snippd-agent (hyphen).

4) Laptop: this workspace has git with an initial commit. Add your remote and push:
      git remote add origin https://github.com/YOUR_ORG/snippd-beta-demo.git
      git branch -M main
      git push -u origin main
